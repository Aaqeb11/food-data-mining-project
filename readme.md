# Smart Nutrition Recommender

A data mining and machine learning approach for personalized dietary planning that helps users achieve nutritional goals such as bulking, slimming, or adhering to dietary constraints (vegan, gluten-free, etc.).

## About

This project develops an intelligent nutrition recommendation system that leverages clustering algorithms, machine learning, and neural networks to deliver personalized dietary guidance. The system analyzes nutritional data from large-scale datasets and aligns food recommendations with individual user goals and dietary restrictions.

## Datasets

- **USDA FoodData Central**: Over 300,000 food items with detailed nutrient profiles including macronutrients and micronutrients
- **Open Food Facts**: 2.5+ million global food entries with ingredients, additives, and allergen information
